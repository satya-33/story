cmd for git version control
"ls -a" to see hidden files
"git status" to see untrack files
" git add filename" & "git add . " to add for tracking 
"git commit -m "commited change "" for comminting the change
"git log" for see the information about the commit 
"git diff filename " to see difference in between current and previous 
" git checkout filename" to undo and go to previous version
"git remote add origin https://github.com/satya-33/story.git" 
"git push -u origin main"
